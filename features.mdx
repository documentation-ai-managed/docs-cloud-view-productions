## Creating Immersive Virtual Tours

Virtual tours transport buyers inside properties without physical visits. You schedule a shoot, and professionals capture high-resolution 360-degree images stitched into interactive experiences.

<Steps>
  <Step title="Schedule Shoot" icon="calendar">
    Select a property and book a certified photographer via the dashboard.
  </Step>
  <Step title="Capture Content" icon="camera">
    Use provided equipment for 360 photos and hotspots.
  </Step>
  <Step title="Process & Publish" icon="upload">
    Upload raw files; automated processing generates the tour in `<24` hours.
  </Step>
  <Step title="Share Everywhere" icon="share-2">
    Embed on your site or share unique links.
  </Step>
</Steps>

<Callout kind="tip">
  Add hotspots with agent notes or floor plan popups to boost interactivity.
</Callout>

## Building Customizable Property Websites

Launch dedicated microsites tailored to each listing. Customize layouts, colors (like brand color `#1598e1`), and content without coding.

<Tabs>
  <Tab title="Basic Site" icon="layout">
    Pre-built templates with photos, tours, and contact info.

    ```
    Example embed for tour:
    <iframe src="https://tours.cloudviewproductions.com/property/123" width="100%" height="500"></iframe>
    ```
  </Tab>
  <Tab title="Advanced Customization" icon="settings">
    Add custom CSS or integrate MLS data.

````jsx
// Custom embed script
const embedTour = (id) => {
  const iframe = document.createElement('iframe');
  iframe.src = `https://tours.cloudviewproductions.com/property/${id}`;
  iframe.width = '100%';
  iframe.height = '600';
  document.getElementById('tour-container').appendChild(iframe);
};
embedTour('123');
````
  </Tab>
</Tabs>

## Tracking Performance with Analytics

Monitor campaign success with dashboards showing views, time spent, and lead forms.

<CodeGroup tabs="Dashboard Embed,API Query">
````javascript
// Embed analytics widget
<script src="https://analytics.cloudviewproductions.com/widget.js"></script>
<div id="cvp-analytics" data-property="123"></div>
````

````javascript
// Fetch analytics via API
const fetchAnalytics = async (propertyId) => {
  const response = await fetch(`https://api.example.com/properties/${propertyId}/analytics`, {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  return response.json();
};
````
</CodeGroup>
